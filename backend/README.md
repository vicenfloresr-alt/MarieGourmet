# Backend Marie Gourmet

## Ejecutar

```bash
mvn spring-boot:run
```

## Pruebas

```bash
mvn test
```

## Rutas

```text
GET    http://localhost:8080/api/productos
POST   http://localhost:8080/api/productos
PUT    http://localhost:8080/api/productos/1
DELETE http://localhost:8080/api/productos/1

GET    http://localhost:8080/api/eventos
POST   http://localhost:8080/api/eventos
PUT    http://localhost:8080/api/eventos/1
DELETE http://localhost:8080/api/eventos/1

POST   http://localhost:8080/api/cotizar
POST   http://localhost:8080/api/lista-compras
```
