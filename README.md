## Descripción del proyecto

Marie Gourmet es un sistema web que permite a clientes corporativos cotizar eventos gastronómicos, calcular automáticamente los insumos necesarios, gestionar el catálogo de productos y generar listas de compra para abastecimiento.

## Casos de uso principales

| ID | Caso de uso | Actor principal |
|----|-------------|-----------------|
| CU-MG-001 | Cotizar evento | Cliente Corporativo |
| CU-MG-002 | Calcular insumos | Administrador |
| CU-MG-003 | Gestionar productos | Administrador |
| CU-MG-004 | Generar lista de compras | Encargado de Compras |

---

## Tecnologías

- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Springboot + API REST
- **Base de datos:** MySQL
- **Infraestructura:** AWS
- **Gestión ágil:** Trello
